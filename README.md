# AfriTalent – Plateforme Freelance Tech Africaine

**Nom :** Oumar  
**Classe :** CyberSécurity 
**Date :** Mai 2026

## Description

AfriTalent est un site vitrine moderne pour une plateforme fictive de mise en relation entre freelances tech africains et entreprises. Le site présente la plateforme, ses fonctionnalités, ses tarifs, des profils de freelances, et invite les visiteurs à s'inscrire. Développé avec les standards web 2026 : design Bento Grid, typographie expressive, dark mode, animations JavaScript et responsive design.

## Technologies utilisées

- **HTML5** — Structure sémantique (`header`, `nav`, `main`, `section`, `footer`)
- **CSS3** — Variables CSS, Flexbox, CSS Grid, Bento Grid, transitions, animations, media queries
- **Bootstrap 5** — Navbar, Cards, Carousel, Accordion, Grid System, Modal
- **JavaScript Vanilla** — DOM manipulation, IntersectionObserver, localStorage, validation de formulaire
- **Bootstrap Icons** — Icônes UI
- **Google Fonts** — Poppins (titres) + Inter (corps)

## Fonctionnalités principales

1. **Dark Mode** — Toggle jour/nuit avec persistance via localStorage
2. **Compteurs animés** — Statistiques qui s'animent au scroll (IntersectionObserver)
3. **Filtrage dynamique** — Catalogue freelances filtrable par catégorie sans rechargement
4. **Validation de formulaire** — Validation complète avec regex email, longueur min, retour visuel
5. **Navbar dynamique** — Style qui change au scroll (fond, ombre, shrink)
6. **Retour en haut** — Bouton flottant avec smooth scroll
7. **Animations fade-in** — Sections apparaissent en fondu au scroll

## Pages du site

| Page | Description |
|------|------------|
| `index.html` | Accueil avec hero, stats, Bento Grid, services, témoignages, CTA |
| `freelances.html` | Catalogue de 9 freelances avec filtres dynamiques |
